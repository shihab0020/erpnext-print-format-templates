# ERPNext Custom Print Format Gallery 🚀

This repository provides ready-to-use, professional HTML/CSS templates for **ERPNext** and **Frappe Framework**. Use these to replace the standard system print formats with high-quality, business-ready designs.

## 🛠 Features
* Fully Responsive Layouts
* Jinja2 powered dynamic fields
* GST/VAT compliant layouts
* Support for custom headers and footers

## 📂 Included Templates
* **Sales Invoice:** (Classic, Modern, and Minimalist styles)
* **Quotation:** (With item images and terms)
* **Delivery Note:** (Compact layout)
* **Purchase Order:** (Professional vendor-facing design)

## 🚀 How to Install
1. Open the folder of the template you want.
2. Copy the code from the `.html` file.
3. In your ERPNext instance, go to **Print Format List** > **New**.
4. Select the **DocType** (e.g., Sales Invoice).
5. Check the **Custom Format** box.
6. Paste the code into the **HTML** section and Save.
7. PDF Generator select to chrome
